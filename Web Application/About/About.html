About Us

This is a simple static website created using HTML, CSS, and JavaScript. It is to set up a basic DevOps pipeline to automatically deploy a static website on AWS using popular DevOps tools. This project will introduce you to key AWS services, CI/CD concepts, and basic website deployment practices. It's a great starting point for beginners to gain hands-on experience with DevOps in the AWS cloud environment. As you become more comfortable with these tools and practices, you can explore more advanced DevOps projects and expand your AWS knowledge.
